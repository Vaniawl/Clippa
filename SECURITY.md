# Security Policy

## Supported versions

Security fixes are provided for the latest published Clippa release.

## Reporting a vulnerability

Do not open a public issue for a security or privacy vulnerability. Use
[GitHub private vulnerability reporting](https://github.com/Vaniawl/Clippa/security/advisories/new)
and include:

- the affected Clippa and macOS versions;
- steps to reproduce the issue;
- the expected security or privacy impact;
- any suggested mitigation.

Do not include real passwords, tokens, or private clipboard history. Reports
will be reviewed before details are made public.
