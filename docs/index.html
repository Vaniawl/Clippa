Fast capture

Text, links, images, and file references appear in one searchable history without opening a full app window.

Keyboard first

Choose your global shortcuts, paste the selected item, and pin important clips for later.

Private by default

Clippa does not send clipboard contents to a server. Local history is encrypted on your Mac.

No cloud clipboard storage

Clippa has no account system, no analytics, no telemetry, and no remote database for your clipboard. The app keeps history only on your Mac so search and paste can work locally.

No uploadsClipboard text, images, links, and files are not uploaded by Clippa.
Local encryptionStored history is encrypted with a Keychain-backed AES-GCM key.
Your controlYou can clear unpinned history or all history from the app settings.

Screenshots

Clippa keeps the daily workflow small: a quick panel for paste, a focused settings screen, and privacy details that are easy to verify.

Install

Run the npx command on macOS. The installer opens Clippa after copying the app. Grant Accessibility only if you want automatic paste.

npx github:Vaniawl/Clippa
npm install -g install-clippa
